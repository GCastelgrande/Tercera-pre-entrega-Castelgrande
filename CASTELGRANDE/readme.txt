Programa hecho con Python y Django para el curso de Coderhouse.
El objetivo es tener un programa que sirva para tener presente tareas a realizar.
Hay que hacer un registro como usuario para poder acceder a la ventana de actividades. Una vez en ella se pueden agregar tareas a realizar,
eliminar tareas hechas, modificarlas o directamente poner un "click" para marcarlas como realizadas.
El admin puede recorrer los usuarios, pudiendo agregar nuevos o quitar.
Como este trabajo tiene la intencia de preparnos para el último proyecto del curso, dejé algunas
cosas de carácter estético para que puedan ser abordadas por el tutor.
